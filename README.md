# astromatch
C++ Zodiac Compatibility Checker
What it does-
Enter two zodiac signs and AstroMatch will tell you your elements (Fire, Earth, Air, or Water) and how compatible you are with each other!
How to run-
Compile and run in your terminal:
g++ astromatch.cpp -o astromatch
./astromatch
Notes-
This is my first ever C++ project and I just wanted to get something out into the world! I know there's a lot of room for improvement (like better input handling, more detailed compatibility results, and a nicer UI), but I'm proud of it as a starting point. More projects to come!
